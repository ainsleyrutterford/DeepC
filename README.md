# CNN architectures for Coral Density Analysis

A simple implementation of 2D SegNet, 2D U-Net and 3D U-Net architectures in Keras.

## Prerequisits

- `numpy<1.17`
- `tensorflow==1.8`
- `keras==2.2.4`

These can be installed by running `pip install -r requirements.txt`

## Usage

```bash
python train.py
```
